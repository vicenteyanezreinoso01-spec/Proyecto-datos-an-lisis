# 🟢 Proyecto 1: Análisis Descriptivo de Ventas Retail

## 🎯 Objetivo
Aplicar limpieza de datos y estadística descriptiva sobre un dataset real de ventas, 
generando insights de negocio a partir de datos crudos y desordenados.

## 🧠 Habilidades a desarrollar
- Detección y tratamiento de valores nulos y duplicados
- Identificación de outliers
- Medidas de tendencia central (media, mediana, moda)
- Medidas de dispersión (desviación estándar, rango intercuartílico)
- Visualización básica (histogramas, boxplots, gráficos de barras)

## 🛠️ Tecnologías
Python, pandas, matplotlib, seaborn, Jupyter/Google Colab

## 📦 Dataset sugerido
https://raw.githubusercontent.com/vicenteyanezreinoso01-spec/Proyecto-datos-an-lisis/refs/heads/main/Proyecto1_Analisis_Descriptivo/retail_store_sales.csv

## 🗺️ Ruta paso a paso

| Paso | Actividad |
|------|-----------|
| 1 | Cargar el dataset y hacer inspección inicial (`.info()`, `.describe()`, `.head()`) |
| 2 | Detectar y tratar nulos/duplicados |
| 3 | Calcular medidas de tendencia central y dispersión por categoría |
| 4 | Visualizar distribuciones (histogramas) de las variables numéricas clave |
| 5 | Identificar y analizar outliers |
| 6 | Redactar 2 conclusiones de negocio basadas en los hallazgos |
