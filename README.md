# 📊 Proyecto-datos-analisis

Repositorio personal de aprendizaje en Data Science, construido como una ruta progresiva 
de proyectos prácticos — desde fundamentos de análisis de datos hasta modelos de 
machine learning e integración de flujos completos.

## 🎯 Objetivo

Construir un portafolio coherente y progresivo que demuestre dominio práctico de las 
principales áreas de Data Science: limpieza de datos, visualización, regresión, 
clasificación, clustering, series temporales y pipelines de datos.

## 🧠 Filosofía de la ruta

1. **Fundamentos antes que modelos** — no hay modelo bueno sin datos limpios y bien entendidos
2. **Supervisado antes que no-supervisado** — primero problemas con "respuesta correcta" para poder autoevaluarme
3. **Un proyecto, una habilidad dominante** — evitar mezclar demasiadas técnicas a la vez
4. **Enfoque de negocio** — cada proyecto conecta con un caso real, no solo código por código
