Proyecto 1: Complemento de Análisis de Datos

Repositorio personal de estudio para reforzar los contenidos del curso **ICS40125 (Data Science)** 
con proyectos prácticos externos, usando Python, pandas y modelos estadísticos.

Objetivo

Conectar la teoría del curso con ejercicios reales de análisis estadístico y modelado,
tomando como referencia el repositorio [Python_Estadistica de JoseAlRic](https://github.com/JoseAlRic/Python_Estadistica).

Ruta de estudio

| Semana | Tema del curso (ICS40125) | Práctica complementaria |
|--------|---------------------------|--------------------------|
| 1 | Repaso de pandas y limpieza de datos | Clonar el repo externo, correr notebook de Análisis Descriptivo |
| 2 | Regresión lineal (Labs) | Comparar StatsModels vs Scikit-Learn con dataset propio |
| 3 | Proyecto final: predicción de churn | Replicar comparación de modelos con dataset ABC Multistate Bank |
| 4 | Cierre y entrega del proyecto | Documentar aprendizajes en este README como portafolio |

conceptos clave a dominar

- [ ] Medidas de tendencia central (media, mediana, moda)
- [ ] Distribución de frecuencias y visualización (histogramas)
- [ ] Medidas de dispersión (desviación estándar, rango intercuartílico)
- [ ] Regresión lineal: análisis gráfico y transformación de variables
- [ ] Comparación de modelos (StatsModels vs Scikit-Learn)
- [ ] Guardado y reutilización de modelos entrenados
